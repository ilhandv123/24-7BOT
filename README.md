# 🤖 ILHANBOT AFK SYSTEM

A simple Minecraft AFK bot that can join servers, stay active, and reconnect automatically.

---

## ✨ Features

- ✅ Automatically joins your server  
- 🔁 Reconnects every 20 seconds if disconnected  
- 🦘 Moves, jumps, and rotates (to stay AFK)  
- 💬 Sends random chat messages  
- ⏱ Restarts every 1 hour  
- 🚫 Changes username only if banned  
- 📡 Keeps trying if server is offline
